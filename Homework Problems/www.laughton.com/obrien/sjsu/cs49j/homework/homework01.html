In this problem you will use various String methods. Complete the class StringsForAll given to you. You will find it at the Codecheck URL below

  • print 4 times the length of the given word (use a string method - do not just print 20 or 24 or ...). Note: There is a String method length() which returns the length of the String
  • print the uppercase version of the word (do not change the original word)
  • replace "s" with "$" Do not change the original word
  • print the changed word and the original word on the same line separated by a space
  • replace "r" with "12" in the changed word
  • replace "a" with "4", in the new changed word

You must use the String replace method to replace characters. No credit for simply printing the word with replacements or replacing the entire word.
